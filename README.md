# 🚀 niL MC Launcher

**Portable Minecraft 1.8.9 Launcher** on Legacy Fabric. Optimized for BedWars & PvP.

## 📖 About

**niL MC Launcher** is a lightweight, portable Minecraft 1.8.9 launcher built on Legacy Fabric. Designed specifically for **Low-End PCs** and optimized for **BedWars** and **PvP** gamemodes.

### ✨ Features

- 📦 **Portable** - USB-ready, extract & play anywhere
- ☕ **Embedded Java** - No Java installation required
- ⚙️ **Custom RAM** - Allocate 1-4GB or custom MB
- 🚀 **Optimized JVM** - Maximum FPS, minimal lag
- 🛡️ **Crash-Free** - Stable and reliable
- ⚔️ **PvP & BedWars Focused** - Optimized for competitive gameplay
- 💻 **Low-End PC Friendly** - Runs smoothly on older hardware

---

## 🎮 How to Use

### Quick Guide

1. **Click** on `launcher.bat` file
2. **Choose** option `[1] Launch Game`
3. **Enter** your Minecraft username
4. **Select** RAM allocation (1-4GB)
5. **Press** Enter and game will be launched!

### Step-by-Step

| Step | Action |
|------|--------|
| 1 | Extract the downloaded archive |
| 2 | Run `launcher.bat` |
| 3 | Select `[1] Launch Game` |
| 4 | Type your username |
| 5 | Choose RAM (1-4GB) |
| 6 | Wait for game to launch 🚀 |
